# dotfiles

### Dependencies
```brew install git```
```brew install hub```
```brew cask install iterm2```
```brew cask install kitty```
```brew install ripgrep```
```brew tap homebrew/cask-fonts && brew cask install font-fira-code```

### Setup
1. ```cd ~```
2. ```git clone <repo url> dotfiles```
3. ```cd dotfiles```
4. ```git submodule update --init --recursive```
5. ```./install.sh```
